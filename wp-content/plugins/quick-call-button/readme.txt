Add quick call button, call now button for any wordpress website on ipad and mobile phone.

== Description ==

Quick Call Button is a plugin add quick call button that helps your customers avoid typing in phone numbers that can be instantly dialed through your website.
Quick Call Button is a powerful development tool that gives you all the options from button placement to custom colors from the administration page.
Show phone numbers or any text on any screen.
Drag and Drop on Desktop.

**Features**

Customize display on-screen varies from cell phone to desktop.
Customize the location of the quick call button anywhere you want.
Customize the background color bar.
Customize button background color.
Customize your phone number.
Customize the display of text on different screens.
Select Images Button.

[Live Demo](http://longvietweb.com/plugins/quick-call-button)
[Plugin Homepage](http://longvietweb.com/plugins/quick-call-button)

== Installation ==

1. Upload the entire 'quick-call-button' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the Settings->Quick Call Button screen to configure the plugin.
